# Asset Attribution

All assets in this project use one of the following licenses:
- **CC0 1.0** (Public Domain)
- **CC-BY 3.0** (Attribution required)
- **CC-BY 4.0** (Attribution required)

## How to Add Credits

When adding a new asset with a CC-BY license, add a row to the table above with:
1. **Asset** — file name or description of the asset
2. **Author** — original creator's name or username
3. **License** — one of: CC0, CC-BY 3.0, CC-BY 4.0
4. **Source URL** — link to the original asset page

CC0 assets do not legally require attribution but should still be listed here for tracking purposes.

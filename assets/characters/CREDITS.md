# Character Art Credits

Attribution for all character sprites shipped in `assets/characters/`. Required under the CC-BY-SA 4.0 / CC-BY / OGA-BY licenses governing LPC assets.

## How this file is maintained

Auto-generated by `tools/lpc-generator/game-batch.mjs` (Playwright run). The batch script saves a per-character `<name>_credits.txt` next to each spritesheet; this Markdown file summarises those TXTs for quick reference.

1. Run `node tools/lpc-generator/game-batch.mjs` (dev server at `localhost:5173` must be running).
2. Re-run the aggregation step documented in `docs/assets/lpc-automation-pipeline.md` (inline Python in "Credits aggregation" section) to refresh this file.
3. The authoritative per-layer attribution (with Notes + Links) lives in each `<name>_credits.txt` — this file is a summary only.
